# 🛍️ Shopify Sales & Customer Funnel Dashboard

An interactive Power BI dashboard analyzing Shopify sales performance, customer behavior, regional trends, and purchase patterns.

---

# 📌 Project Overview

This project provides a complete business intelligence solution for analyzing Shopify e-commerce data using Power BI.  
The dashboard focuses on sales insights, customer segmentation, customer funnel analysis, regional performance, and product category trends.

The report is designed with a modern dark-themed UI and interactive analytics features to help businesses monitor KPIs and customer activities effectively.

---

# 🎯 Objectives

- Analyze overall sales performance
- Track customer purchasing behavior
- Identify repeat and single-order customers
- Monitor customer activity trends
- Discover top-performing product categories
- Analyze regional sales distribution
- Understand gateway/customer distribution
